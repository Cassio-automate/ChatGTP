# Finance Site

Exemplo simples de controle de gastos pessoais em uma página web.

## Funcionalidades
- Registrar despesas com descrição e valor
- Definir meta de gastos
- Lista de despesas
- Sugestões básicas de economia baseadas nos lançamentos

## Como usar
Basta abrir o arquivo `index.html` em um navegador moderno. Não é necessário instalar dependências.
