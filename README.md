# Financeiro App

Este repositório contém um aplicativo simples de controle financeiro pessoal escrito em Python utilizando o framework Kivy.

## Requisitos

- Python 3
- Dependências listadas em `requirements.txt`

Instale as dependências com:

```bash
pip install -r requirements.txt
```

## Executar

Para iniciar a aplicação execute:

```bash
python main.py
```

O aplicativo cria um banco SQLite `finance.db` localmente para armazenar usuários e transações.
