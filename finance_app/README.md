# Finance App

Aplicativo simples para controle de gastos pessoais com sugestões automáticas.

## Funcionalidades

- Adicionar despesas com descrição e valor
- Definir uma meta de gastos
- Lista de despesas registradas
- Algoritmo simples para sugerir reduções e avisar se a meta for excedida

## Como executar

1. Instale o [Node.js](https://nodejs.org) e o Expo (`npm install -g expo-cli`).
2. Execute `npm install` para instalar as dependências.
3. Inicie o aplicativo com `npm start`.
4. Utilize o aplicativo em um emulador ou celular via aplicativo Expo.

Este projeto é apenas um exemplo básico de como iniciar um aplicativo de controle financeiro com **React Native** e sugestões de IA simples.
